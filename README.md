# Food and Things

En enkel React/TailwindCSS-butik med varukorg, produktdetaljer och kassa.

---

## 🛠️ Installation

1. **Klona repot**

   ```bash
   git clone <repo-url>
   cd my-app
   ```

2. **Installera beroenden**

   ```bash
   npm install
   ```

3. **Starta utvecklingsservern**

   ```bash
   npm run dev
   ```

4. **Öppna i webbläsaren**
   Besök [http://localhost:5173] (eller den port som Vite visar).
